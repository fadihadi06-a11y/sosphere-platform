# 🆘 World SOS - منصة السلامة العالمية

## نظرة عامة

**World SOS** هي منصة سلامة وأمان عالمية متكاملة تتضمن ثلاث واجهات رئيسية:

### 🎯 المنصات الثلاث

#### 📱 1. تطبيق الموبايل
واجهة المستخدم النهائي مع:
- زر SOS كبير للطوارئ
- 5 سيناريوهات (خطر، طبي، حادث، مطاردة، تائه)
- رحلات آمنة مع تتبع
- إدارة جهات الاتصال
- سجل الطوارئ

#### 🏢 2. لوحة تحكم الشركة
إدارة سلامة الموظفين مع:
- مراقبة الطوارئ المباشرة
- KPI Cards
- جدول الموظفين
- غرفة التحكم
- التقارير والبث

#### 👑 3. لوحة المالك
التحكم الشامل في المنصة:
- إدارة الشركات
- إحصائيات عالمية
- مراقبة النظام
- الإيرادات

## 📱 شاشات تطبيق الموبايل

### 1. الصفحة الرئيسية (Home)
- زر SOS دائري كبير (160px)
- تأثير Pulse مستمر
- 5 أزرار سيناريوهات
- 3 بطاقات مميزات
- شريط تنقل سفلي

### 2. طوارئ نشطة (Active Emergency)
- خلفية حمراء متحركة
- عداد الوقت
- حالة الإجراءات:
  - ✓ SMS sent
  - ✓ Location shared
  - ⏳ Calling contact
- خريطة مباشرة
- زر "أنا بخير"

### 3. رحلة آمنة (Safe Trip)
- إدخال الوجهة
- اختيار المدة
- تحديد جهات الاتصال
- خريطة مباشرة
- عداد تنازلي
- زر "وصلت بأمان"

### 4. جهات الاتصال (Contacts)
- قائمة الجهات
- تفعيل/تعطيل SMS
- تفعيل/تعطيل الاتصال
- إضافة/تعديل/حذف

### 5. السجل (History)
- تبويبات التصفية (الكل، طوارئ، رحلات)
- تفاصيل كل حالة
- الموقع والوقت
- حالة الإنجاز

### 6. الحساب (Profile)
- معلومات المستخدم
- الإعدادات
- تبديل اللغة
- المظهر الداكن
- تسجيل الخروج

## 🏢 لوحة تحكم الشركة

### نظرة عامة (Overview)
- **KPI Cards:**
  - إجمالي الموظفين
  - نشط اليوم
  - المهام الخارجية
  - نقاط السلامة

- **Emergency Alerts:**
  - تنبيهات فورية
  - عداد تلقائي للاتصال
  - معلومات الموقع
  - إجراءات سريعة

- **جدول الموظفين:**
  - الصورة والاسم
  - الحالة (طوارئ، نشط، مهمة، غير نشط)
  - مستوى المخاطر
  - القسم
  - الهاتف

### غرفة التحكم (Control Room)
- **شريط التنبيه:**
  - أخضر (كل شيء آمن)
  - أحمر (طوارئ نشطة)
  - عداد الوقت

- **3 أعمدة:**
  1. قائمة الحالات النشطة
  2. خريطة مباشرة
  3. تفاصيل الحالة

- **إجراءات:**
  - اتصال فوري
  - عرض الموقع
  - تحديد كمحلول

### نظرة عامة (Overview)
